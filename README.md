# SenseNote Android App

## Introduction

SenseNote is an Android application developed to act as a digital assistant supporting sensory regulation and specific behavior management for children in inclusive education, particularly those with ASD or ADHD. 

This repository contains the Android frontend application built with Kotlin and Jetpack Compose, focusing on modern Android development practices, responsive layouts, and REST API integration.

The project aims to help educators seamlessly manage smart seat maps, identify sensory triggers, and log ABC (Antecedent-Behavior-Consequence) behavior records efficiently in real-time, significantly reducing cognitive load during classroom management.
